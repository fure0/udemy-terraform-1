# 35. 서브넷 작성

`aws_subnet` 리소스를 정의할 때 자주 쓰는 인자입니다. 강의 슬라이드 표를 옮겨 적었습니다.

| 항목 | 타입 | 설명 |
| :--- | :--- | :--- |
| **vpc_id** * | string | 서브넷이 속할 VPC의 ID |
| **availability_zone** | string | 가용 영역 (예: `ap-northeast-1a`) |
| **cidr_block** * | string | 서브넷의 IPv4 CIDR 블록 (**VPC CIDR 안에 포함**되어야 함) |
| **map_public_ip_on_launch** | bool | 이 서브넷에 띄운 인스턴스에 퍼블릭 IPv4를 자동 할당할지 여부 |
| **tags** | object(map) | Name, Type 등 태그 |

`*` 는 슬라이드 기준 **필수 인자**입니다.

## CIDR 주의

서브넷의 `cidr_block`은 **반드시 해당 VPC의 `cidr_block` 범위 안**이어야 합니다. VPC가 `10.0.0.0/20`인데 서브넷만 `192.168.x.0/24`로 두면 AWS에서 `InvalidSubnet.Range` 오류가 납니다.

## 실습 코드 위치

서브넷 정의는 **`terraform/network.tf`** 의 `aws_subnet` 블록들에 있습니다.

```23:77:terraform/network.tf
resource "aws_subnet" "public_subnet_1a" {
  vpc_id                  = aws_vpc.vpc.id
  availability_zone       = "ap-northeast-1a"
  cidr_block              = "192.168.1.0/24"
  map_public_ip_on_launch = true

  tags = {
    Name    = "${var.project}-${var.environment}-public-subnet-1a"
    Project = var.project
    Env     = var.environment
    Type    = "public"
  }
}

resource "aws_subnet" "public_subnet_1c" {
  vpc_id                  = aws_vpc.vpc.id
  availability_zone       = "ap-northeast-1c"
  cidr_block              = "192.168.2.0/24"
  map_public_ip_on_launch = true

  tags = {
    Name    = "${var.project}-${var.environment}-public-subnet-1c"
    Project = var.project
    Env     = var.environment
    Type    = "public"
  }
}

resource "aws_subnet" "private_subnet_1a" {
  vpc_id                  = aws_vpc.vpc.id
  availability_zone       = "ap-northeast-1a"
  cidr_block              = "192.168.3.0/24"
  map_public_ip_on_launch = false

  tags = {
    Name    = "${var.project}-${var.environment}-private-subnet-1a"
    Project = var.project
    Env     = var.environment
    Type    = "private"
  }
}

resource "aws_subnet" "private_subnet_1c" {
  vpc_id                  = aws_vpc.vpc.id
  availability_zone       = "ap-northeast-1c"
  cidr_block              = "192.168.4.0/24"
  map_public_ip_on_launch = false

  tags = {
    Name    = "${var.project}-${var.environment}-private-subnet-1c"
    Project = var.project
    Env     = var.environment
    Type    = "private"
  }
}
```

위 VPC는 같은 파일의 `aws_vpc.vpc`(`cidr_block = "192.168.0.0/20"`)에 연결됩니다. 퍼블릭 서브넷은 `map_public_ip_on_launch = true`, 프라이빗은 `false` 로 두는 패턴이 일반적입니다.
