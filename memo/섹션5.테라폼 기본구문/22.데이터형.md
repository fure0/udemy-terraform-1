## HCL2에서 다루는 데이터형

Terraform(HCL2)에서 변수·식에 쓸 수 있는 타입을 크게 **기본형**, **구조체**, **컬렉션**으로 나눈다.

| 구분 | 타입 이름 | 설명 |
|------|-----------|------|
| **기본형** | `string` | 유니코드 문자열 |
| | `number` | 수치(정수·소수 모두) |
| | `bool` | `true` / `false` |
| **구조체** | `object({ … })` | 키마다 타입이 정해진 키-값 구조 |
| | `tuple([ … ])` | 위치(인덱스)마다 타입이 정해진 고정 길이 배열 |
| **컬렉션** | `list(<TYPE>)` | 요소가 모두 같은 타입인 목록 |
| | `map(<TYPE>)` | 키는 문자열, 값은 지정한 타입으로 통일 |
| | `set(<TYPE>)` | 같은 타입 요소만 모으며, **중복 값 제거** |

---

### 기본형 (Primitive)

가장 자주 쓰는 단일 값 타입이다.

```terraform
variable "message" {
  type    = string
  default = "Hello World"
}

variable "max_count" {
  type    = number
  default = 10
}

variable "is_enable" {
  type    = bool
  default = true
}
```

- `string`: 큰따옴표로 문자열 표현.
- `number`: 따옴표 없이 숫자만.
- `bool`: `true` / `false` 키워드(따옴표 없음).

---

### object

키-값 형태이며, **각 키의 타입을 미리 스키마로 정의**한다. 서로 다른 타입을 한 변수에 넣을 수 있다.

```terraform
variable "obj_sample" {
  type = object({
    name = string
    age  = number
  })
  default = {
    name = "tanaka"
    age  = 28
  }
}
```

접근: `var.obj_sample.name` 처럼 **점(.)**으로 필드 지정.

---

### tuple

**N번째 요소가 어떤 타입인지**가 순서대로 고정된 배열. 길이와 위치별 타입이 정해진다.

```terraform
variable "tuple_sample" {
  type = tuple([
    string,
    number,
  ])
  default = ["tanaka", 28]
}
```

접근: `var.tuple_sample[0]` 처럼 **인덱스**(0부터).

---

### list

**모든 요소가 같은 타입**인 배열.

```terraform
variable "list_sample" {
  type    = list(string)
  default = ["tanaka", "sato"]
}
```

접근: `var.list_sample[0]`.

---

### map

**키는 항상 문자열**, 값은 `map(값타입)`에 맞춰 통일한다.

```terraform
variable "map_sample" {
  type = map(string)
  default = {
    "High" = "m5.2xlarge"
    "Mid"  = "m5.large"
    "Low"  = "t2.micro"
  }
}
```

접근: `var.map_sample.High` 또는 `var.map_sample["High"]`.

---

### set

같은 타입의 모음이지만, **값의 중복이 제거**된다. `default`에 같은 값을 여러 번 넣어도 최종적으로는 유일한 값만 남는다.

```terraform
variable "set_sample" {
  type = set(string)
  default = [
    "tanaka",
    "sato",
    "tanaka",
    "sato",
  ]
}
```

`for` 식 등으로 순회할 수 있다.

```terraform
[for itm in var.set_sample : itm]
```
