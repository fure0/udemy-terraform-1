## 변수
### locals
- 로컬변수
- 프라이빗 변수라서 외부에서 변경 불가


```terraform
locals {
  project = "tastylog"
  env     = "dev"
}

resource <RESOURCE_TYPE> <RESOURCE_NAME> {
  ...
  tags = {
    Name  = "${local.project}-${local.env}-vpc"
  }
  ...
}

output <OUTPUT_TYPE> {
  ...
}
```

### variables
- 외부에서 변경가능
- 커맨드라인 실행시 옵션이나 파일지정으로 덮어쓰기 가능

```terraform
variable "project" {
  type    = string
  default = "tastylog"
}

resource <RESOURCE_TYPE> <RESOURCE_NAME> {
  ...
  tags = {
    Name = "${var.project}-dev-vpc"
  }
  ...
}

output <OUTPUT_NAME> {
  ...
}
```
