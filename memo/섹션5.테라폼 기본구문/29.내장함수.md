## Terraform 내장 함수

Terraform(HCL2)에는 값을 가공할 수 있는 내장 함수가 여러 카테고리로 준비되어 있다.

| 종류 | 설명 |
|------|------|
| `numeric` | 수치 연산(절대값, 올림/내림, 최대/최소 등) |
| `string` | 문자열 처리(포맷, 결합, 추출 등) |
| `collection` | 리스트/맵/셋 등 컬렉션 연산(결합, 분할, 정렬 등) |
| `encoding` | Base64, CSV, JSON, YAML 등 인코딩/디코딩 변환 |
| `filesystem` | 파일/경로 관련 처리(파일 읽기, 디렉터리명 등) |
| `date & time` | 현재 시각, 날짜/시간 포맷 처리 |
| `Hash & Crypt` | 해시/암호화 관련 함수(Bcrypt, MD5, SHA-256 등) |
| `IP Network` | CIDR 연산, 호스트/서브넷 관련 계산 |
| `Type Conversion` | 타입 변환(`bool`, `string`, `number` 등) |

---

## 내장 함수 테스트 방법 (`terraform console`)

내장 함수는 `terraform console`에서 바로 시험해 볼 수 있다.

```bash
terraform console [-var 'KEY=VALUE'] [-var-file=VAR_FILE]
```

- `-var <KEY>=<VALUE>`: 변수 값을 직접 넘겨 실행
- `-var-file <FILE>`: 변수 파일(`.tfvars`)을 읽어서 실행

> 변수를 참조하는 식을 테스트할 때는, 해당 변수 정의가 들어 있는 `.tf` 파일이 있는 디렉터리에서 실행하는 것이 안전하다.

### 콘솔 예시

```text
$ terraform console
> floor(4.9)
4
> substr("hello world", 1, 4)
"ello"
```
