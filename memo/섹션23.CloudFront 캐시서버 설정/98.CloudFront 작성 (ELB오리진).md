# 98. CloudFront 작성 (ELB 오리진)

**동적 파일(앱)** 은 **ALB(ELB)** 를 CloudFront **오리진**으로 두고, **기본 동작(default behavior)** 에서 ELB로 전달한다.  
97번 인자 개요를 **`custom_origin_config`** 조합으로 **첫 CloudFront 배포**를 만든다.

슬라이드: **CloudFrontの作成/設定**

1. **CloudFront 생성**
   - **오리진에 ELB 설정**
   - **デフォルトビヘイビア(기본 동작)에 ELB 설정**
2. **동작 확인**

이전: **97 CloudFront 리소스** · 다음: **99 S3 오리진** (`/public/*`)

---

## 목표 (슬라이드·아키텍처)

```
사용자 ──https──→ CloudFront
                      │
         ┌────────────┴────────────┐
         │                         │
    /public/* (99)            デフォルト (*)
         │                         │
        S3                        ELB (ALB)
   (정적 파일)                  (動的ファイル)
```

| 항목 | 98에서 하는 일 |
|------|----------------|
| **오리진** | **`aws_lb.alb`** — `custom_origin_config` |
| **기본 동작** | **패턴 `*` (default)** → ELB · **캐시 없음** |
| **S3 `/public/*`** | **99번** — `ordered_cache_behavior` 로 추가 |

슬라이드: **動的ファイルはELBへ** — 경로 **デフォルト (`*`)** · 캐시 **(なし)**.

## 97번과의 대응

| 슬라이드 5블록 | 98에서 채우는 값 |
|----------------|------------------|
| **기본 설정** | `enabled`, `comment`, `is_ipv6_enabled`, `price_class` |
| **オリジン** | ELB `domain_name` + **`custom_origin_config`** |
| **ビヘイビア** | **`default_cache_behavior`** → `target_origin_id` = ELB |
| **アクセス制限** | `geo_restriction { restriction_type = "none" }` |
| **証明書** | 1단계: **CF 기본 URL** 또는 `aliases` + **`virginia_cert`** |

**OAI·S3 policy** — **99번**. 98은 **ELB 오리진만** 추가해도 됨.

---

## 오리진 — ELB (`custom_origin_config`)

| 항목 | 이 프로젝트 |
|------|-------------|
| **`domain_name`** | `aws_lb.alb.dns_name` (82·84 ALB) |
| **`origin_id`** | 예: `"alb-origin"` — **behavior가 참조** |
| **`origin_protocol_policy`** | 강의: **`"https-only"`** (ALB 443·92 HTTPS) |
| **`origin_ssl_protocols`** | `["TLSv1.2"]` |
| **`http_port`** | `80` |
| **`https_port`** | `443` |

`origin_id` 는 **behavior의 `target_origin_id`와 문자열 일치** (97 슬라이드).

---

## 기본 동작 — 동적·캐시 없음

| 항목 | 98 (ELB·동적) |
|------|----------------|
| **`path_pattern`** | **없음** (`default_cache_behavior` 전용 규칙) |
| **`target_origin_id`** | ELB의 **`origin_id`** |
| **`allowed_methods`** | `GET`, `HEAD`, `OPTIONS`, `PUT`, `POST`, `PATCH`, `DELETE` (앱 API) |
| **`cached_methods`** | `GET`, `HEAD` (동적은 사실상 **캐시 안 함**) |
| **`viewer_protocol_policy`** | **`"redirect-to-https"`** 또는 `"https-only"` |
| **TTL** | **`min_ttl` / `default_ttl` / `max_ttl` = 0** → **캐시 없음** (슬라이드 なし) |
| **`forwarded_values`** | 쿠키·쿼리·헤더 **전달** (세션·API) |

동적 오리진은 **캐시 비활성**이 일반적. 99의 S3 정적 경로만 **캐시 ON**.

## HCL 예 (`cloudfront.tf` — ELB만)

```hcl
resource "aws_cloudfront_distribution" "app" {
  enabled             = true
  is_ipv6_enabled     = true
  comment             = "${var.project}-${var.environment}-cloudfront"
  default_root_object = ""
  price_class         = "PriceClass_All"

  origin {
    domain_name = aws_lb.alb.dns_name
    origin_id   = "alb-origin"

    custom_origin_config {
      http_port              = 80
      https_port             = 443
      origin_protocol_policy = "https-only"
      origin_ssl_protocols   = ["TLSv1.2"]
    }
  }

  default_cache_behavior {
    allowed_methods  = ["DELETE", "GET", "HEAD", "OPTIONS", "PATCH", "POST", "PUT"]
    cached_methods   = ["GET", "HEAD"]
    target_origin_id = "alb-origin"

    forwarded_values {
      query_string = true
      headers      = ["*"]

      cookies {
        forward = "all"
      }
    }

    viewer_protocol_policy = "redirect-to-https"
    min_ttl                = 0
    default_ttl            = 0
    max_ttl                = 0
    compress               = true
  }

  restrictions {
    geo_restriction {
      restriction_type = "none"
    }
  }

  viewer_certificate {
    cloudfront_default_certificate = true
  }
}
```

### 커스텀 도메인 + HTTPS (이후·강의)

`aliases` + **`virginia_cert`** (93):

```hcl
  aliases = ["dev.${var.domain}"]

  viewer_certificate {
    acm_certificate_arn      = aws_acm_certificate.virginia_cert.arn
    ssl_support_method       = "sni-only"
    minimum_protocol_version = "TLSv1.2_2021"
  }
```

Route53 Alias → `aws_cloudfront_distribution.app.domain_name` (87 패턴).

| 경로 | 오리진 | 강의 |
|------|--------|------|
| **`/public/*`** | S3 static | **99** — `ordered_cache_behavior` + OAI |

## 주의

| 항목 | 내용 |
|------|------|
| **ALB 보안 그룹** | CloudFront → ALB는 **인터넷 경유** — SG가 **443/80 허용** (82) |
| **`tokyo_cert` ≠ CloudFront** | 뷰어 인증서는 **`virginia_cert`** (93) |
| **`forwarded_values`** | provider·강의 버전에 따라 **managed policy** 권장 — plan 에러 시 97 참고 |
| **캐시 0** | POST 등은 애초에 캐시 대상 아님 — **동적은 Miss 정상** |
| **헤더 `["*"]`** | 강의 예 — 운영은 필요 헤더만 whitelist |

## 정리

- **98** = CloudFront **생성** + **ELB 오리진** + **default behavior → ELB** + **캐시 없음**.
- **`origin_id`** ↔ **`target_origin_id`** 연결 (97).
- 슬라이드 **動的ファイルはELBへ** = **default `*`**.
