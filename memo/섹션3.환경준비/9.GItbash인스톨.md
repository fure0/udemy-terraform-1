## Windows만 대응, MacOS는 대응 필요없음

## GitBash란
- git + bash
- git을 사용할수 있는 Bash
- BASH란 shell의 한 종류. sh의 진화계, 그밖에는 csh, zsh등이 있음
- Terraform을 Windows에서 동작시키기 위해선 GitBash만 동작확인 됨


## git 다운로드
- 홈페이지에서 OS에 맞춰서 다운로드
- 인스톨시에 다른건 디폴트로 괜찮은데 개행스타일은 Checkout as-is, commit as-is가 추천. 즉 git이 개행코드를 판별하는것이 아닌 있는 그대로

## git 초기화
```bash
git version # git 설치 확인
git config --global user.email "자신의 작업 이메일"
git config --global user.name "자신의 활동명"

git config --global --list # 등록 확인
```
