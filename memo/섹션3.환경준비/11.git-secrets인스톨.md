## git-secrets 인스톨 (Windows)

### 1. git-secrets 설치 (PowerShell)

```powershell
Set-ExecutionPolicy RemoteSigned

git clone https://github.com/awslabs/git-secrets.git
cd ./git-secrets
./install.ps1
```

### 2. git-secrets 초기화 (PowerShell)
```powershell
git secrets --register-aws --global
git secrets --install $env:USERPROFILE/.git-templates/git-secrets
git config --global init.templatedir $env:USERPROFILE/.git-templates/git-secrets
```

### 3. 동작 확인 (Git Bash)
```bash
git init
git add .
git commit -m "test"
```


## git-secrets 인스톨 (macOS)

### 1. git-secrets 설치

```bash
brew install git-secrets
```

### 2. git-secrets 초기화
```bash
git secrets --register-aws --global
git secrets --install ~/.git-templates/git-secrets
git config --global init.templatedir ~/.git-templates/git-secrets
```

### 3. 동작 확인
- test 폴더에 credential 파일을 넣고 커밋하려고 하면 실패되면 정상
```bash
git init
```
